Using a License File

IMPORTANT: ListenUp comes with a "FreeMode" license . It will cause ListenUp to show a brief advertisement for our site the first time the Applet is run. If you want to eliminate the advertisement and support the future development of ListenUp then you can purchase a replacement license file.

If you purchase a license file then you will need to install it on your server and replace the existing "FreeMode" license.

1) Install ListenUp License File

Place the "listenup_license.txt" file in the "{your website path}/listenup/codebase" folder on your web server, next to the JAR files. The Applet looks in the CODEBASE so if you change your CODEBASE in the <APPLET> tag then be sure to copy the license file to the new CODEBASE folder.

Note that at one time we used a file called "listenup.lic" but that caused problems on IIS servers so we switched to a ".txt" file.
