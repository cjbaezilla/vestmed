Converting WAV Files to MP3 Files on your Server

Note that MP3 is a patented codec so we cannot include an MP3 encoder with ListenUp. Your use of MP3 may require royalty payments to the patent holders.

You might wish to upload voice recordings as Speex or WAV files and then distribute them as MP3. This allows you to use standard media players like WinAmp for playback.

Converting a WAV files to MP3 requires an MP3 Encoder. We recommend the LAME encoder. LAME is very easy to build from source on Linux and Mac OS X. Precompiled binary versions are available for Windows.

  • Windows binary versions of LAME
  • LAME Project Home Page for source code.

All the MP3 encoders we know of require WAV files as input. If you are uploading in Speex format then you can convert the Speex to WAV using the instructions on the previous page.

It can take a while to encode an MP3 file. We, therefore, recommend running the encoder in a background process.

I am sorry but we cannot answer questions about how to use the MP3 Encoders. Good documentation is on the LAME websites.
